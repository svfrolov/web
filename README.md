# Проект по веб-разработке

Этот репозиторий содержит проекты по веб-разработке.

## Структура репозитория

- [l1](https://github.com/svfrolov/web/tree/l1) - Лабораторная работа №1: Веб-приложение "Недвижимость"
- [l2](https://github.com/svfrolov/web/tree/l2) - Лабораторная работа №2: Расширенное веб-приложение
- [l3](https://github.com/svfrolov/web/tree/l3) - Лабораторная работа №3: REST API

## Описание проекта

Репозиторий содержит учебные проекты по разработке веб-приложений, выполненные в рамках курса "Разработка интернет-приложений" в МГТУ.

### Лабораторная работа №1 (ветка l1)

Веб-приложение для компании, занимающейся недвижимостью. Реализовано на Django с использованием HTML, CSS.

### Лабораторная работа №2 (ветка l2)

Расширенное веб-приложение с улучшенным интерфейсом и функциональностью.

### Лабораторная работа №3 (ветка l3)

Реализация REST API для веб-приложения с использованием Django REST Framework.

### Лабораторная работа №4 (ветка l4)

Расширенный REST API с авторизацией и разграничением прав доступа.

### Лабораторная работа №5 (ветка l5)

Разработка базового SPA на React с использованием TypeScript. Реализованы компоненты: навигационная панель, карточки услуг, фильтрация, детальные страницы. Настроена работа с API и mock-данными.
